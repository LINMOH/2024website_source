# 算法入门经典 例题 1-3


算法是思想的体操，竞赛是智慧的碰撞。

## 题目

输入两个整数 a 和 b，交换二者的值，然后输出。

## 解答

这一个例题甚至比上一个简单，解法直接使用经典的“三变量法”就行了。

## 解答

```c++
#include <iostream>
using namespace std;

int main()
{
    int a,n,m;
    cin >> n >> m;
    
    a = n;
    n = m;
    m = a;

    cout << n << " " << m;
}
```
